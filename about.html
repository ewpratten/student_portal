About

This is a webapp built by Evan Pratten as an alternative to the Official TVDSB Student Portal

All personal info is kept safe while using the app, as long as you are connected with a secure and trustworthy internet connection (an obfuscated version of your login info may be visible to attachers on an insecure network). Your login token and all info displayed in the My Information page are the only things stored anywhere. This info is kept in your browser's cookies, and cleared when you log out. No information is logged on my API server ever (and all data on the server is wiped every few hours)
